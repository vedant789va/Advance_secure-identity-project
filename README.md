# Advance_secure-identity-project
# Intelligent System for Secure Identity and Transactions

An advanced classroom-ready web prototype that demonstrates:
- Secure digital identity enrollment
- Adaptive trust scoring (biometric + device + email profile)
- Smart transaction risk analysis
- Decision engine (Approved / Manual Review / Blocked)
- Immutable audit-chain style event logging
- Visual dashboard for fraud-risk distribution

## Project Files
- `index.html` - UI layout and sections
- `styles.css` - modern responsive styling and animations
- `app.js` - full working logic for identity, transactions, scoring, and audit

## How to Run
1. Open `index.html` in your browser.
2. Click **Load Demo Data** for instant sample output.
3. Or manually:
   - Create a user in **Secure Identity Enrollment**
   - Process transactions in **Smart Transaction Screening**
4. Show live updates in the dashboard cards, risk chart, and audit chain.

No installation required.

## Presentation Flow (2 Minutes)
1. **Problem Statement**
   Traditional systems verify identity once, but do not continuously assess risk in transactions.
2. **My Solution**
   This intelligent system combines identity trust scoring with transaction-level fraud analytics.
3. **Live Demo**
   - Enroll a user (token + trust score generated)
   - Submit a normal transaction (gets approved)
   - Submit a risky transaction (manual review or blocked)
4. **Highlight Advanced Features**
   - Adaptive scoring model
   - Real-time decision engine
   - Tamper-evident audit chain
   - Dashboard analytics for monitoring
5. **Future Scope**
   Integrate real OTP, face recognition API, and blockchain-backed audit storage.
